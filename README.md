# Climate Action Tracker System 🌍

## SDG 13: Climate Action

This project is a Java Object-Oriented Programming application that helps users track their daily carbon emissions based on:

- Transportation
- Electricity usage
- Waste production

## Features
- Calculates carbon footprint
- Provides eco-friendly suggestions
- Uses OOP concepts (Encapsulation, Inheritance, Polymorphism, Abstraction)
- Uses ArrayList and File Handling

## Technologies
- Java
- OOP Concepts
